Помощ / Настройки на трансформацията

Как работи

Полето Настройки (key=value) ви дава пълен контрол върху преобразуването – формат, подравняване, преименуване на тагове и др. Най-честите параметри са събрани в таблицата по-долу.

Основни опции

  • inputformat json | yaml | xml | csv | emmet | auto
  • outputformat json | yaml | xml | csv | emmet
  • savetohistory true | false
  • align true | false
  • case upper | camel | snake | none

Замяна на тагове / стойности

  • replace.tag.old new
  • replace.val.old new

Примерна конфигурация

inputformat=yaml
outputformat=json
savetohistory=true
align=true
case=snake
replace.tag.version=ver
replace.val.1.0=latest
